# 模板：工作思路/措施/方案类

## Step 1 前置提问

1. **基础信息**
   - 单位全称 / 具体职务 / 负责领域 / 主要职责
   - 材料类型：工作思路 / 实施方案 / 推进措施 / 专项计划
   - 适用场景：年度工作部署 / 专项工作启动 / 问题整改方案 / 创新工作推进
   - 字数要求
   - 上级要求或政策依据

2. **素材重点**
   - 政策背景和上级要求（明确工作依据和目标导向）
   - 现状基础和存在问题（具体数据、典型案例、调研情况）
   - 可行措施和创新做法（具体操作路径和保障条件）

3. **结构框架**
   - 总体目标 → 存在问题 → 主要措施 → 保障措施

## Step 2 大纲模板

```
一、总体目标（包含背景意义、核心目标、预期效果）
二、存在问题（包含现状基础、主要短板、面临挑战）
三、主要措施（包含3-5项具体措施）
四、保障措施（包含组织、制度、督查等保障）
```

约束：措施要针对问题，保障要具体可行。

## Step 3 分段生成策略

| 部分 | 投喂内容 | 说明 |
|------|---------|------|
| 第一部分（总体目标） | 大纲+背景信息（单位性质+政策背景+核心目标） | 体现政策背景和上级要求 |
| 第二部分（存在问题） | 背景信息（含现状问题细节） | 客观分析，问题具体准确 |
| 第三部分（主要措施） | 背景信息+已生成前两部分 | 措施针对问题，可操作 |
| 第四部分（保障措施） | 已生成前三部分 | 组织/制度/人员/经费/督查 |

## Step 4 标题风格

务实、行动导向，体现执行力和决心。
