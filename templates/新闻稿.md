# 模板：新闻稿

非完整五步法流程，通常直接生成初稿后微调即可。

### 活动类新闻稿

```
【角色】请你扮演一名资深机关宣传干事，擅长撰写各类活动新闻稿。

【任务】撰写一篇关于[活动名称]的新闻稿。

【要求】
1. 标题：突出活动主题与亮点
2. 导语：包含时间、地点、活动名称、主办单位、出席重要领导
3. 主体：提炼主要领导讲话要点，介绍活动流程与亮点环节
4. 结尾：说明活动参与范围与影响
5. 语言：简洁明快，符合官方宣传口径
6. 字数：500字左右

【素材】
"""
粘贴活动方案、领导讲话要点、现场亮点等素材
"""
```

### 培训/会议类新闻稿

```
【角色】请你扮演单位内部通讯员的角色，熟悉会议报道规范。

【任务】撰写一篇关于[会议/培训名称]的新闻稿。

【要求】
1. 导语：时间、地点、会议目的、主持及出席领导、主讲人
2. 主体：准确提炼会议核心精神或培训重点内容
3. 结尾：参会人员规模与后续落实要求
4. 注意：首次出现的人名、职务、单位用全称
5. 字数：约500字

【素材】
"""
粘贴会议通知、议程、主持词、讲话稿核心内容
"""
```

### 获奖/成果类新闻稿

```
【角色】请你扮演企业品牌宣传负责人，擅长撰写获奖新闻。

【任务】撰写一篇关于[单位/项目]荣获[奖项名称]的新闻稿。

【要求】
1. 标题：直接点明获奖事实，体现荣誉层级
2. 导语：获奖时间、奖项全称及评选背景
3. 主体：详细介绍获奖项目/成果的价值与创新点
4. 结尾：回顾单位在该领域的长期努力，展望未来
5. 风格：客观自豪，避免过度夸张
6. 字数：400-600字

【素材】
"""
粘贴奖项介绍、项目说明、相关背景资料
"""
```

### 调研/检查类新闻稿

```
【角色】请你扮演随行记者，负责撰写领导调研检查新闻。

【任务】撰写一篇关于[领导职务]调研[地点/单位]的新闻稿。

【要求】
1. 导语：时间、领导姓名职务、调研地点与主要目的
2. 主体：描述现场考察情况，重点提炼领导的指示和要求
3. 结尾：列出陪同调研人员，可简述被调研单位的表态
4. 政治性：表述准确，与上级精神保持一致
5. 字数：500字左右

【素材】
"""
粘贴调研行程、领导现场讲话要点、被调研单位汇报材料
"""
```
