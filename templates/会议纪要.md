# 模板：会议纪要

非五步法流程。会议纪要写作分两步：录音转文字 → AI 生成初稿。

**方法一（推荐）：有优秀纪要定稿，参考范文风格生成**

```
【角色】
你是一位经验丰富的笔杆子，擅长参照优秀范本生成会议纪要。

【任务】
请严格参照【历史范本】的格式、文风和逻辑，结合【本次会议素材】，
撰写新的《关于[会议名称]的会议纪要》。

【核心要求】
1. 参考格式：标题、段落结构、固定用语须与范本一致
2. 内容填充：将本次会议的真实内容按范本逻辑层次填充
3. 风格统一：保持相同的语言严肃性和精炼度

【历史范本】
"""
粘贴一份过往的优秀纪要定稿
"""

【本次会议素材】
"""
会议文字稿
"""
```

**方法二：无优秀定稿，按会议类型选择模板**

### 决策部署型

```
【角色】你是一位资深办公室秘书，专写决策部署型纪要。

【任务】基于以下信息，撰写《关于[会议名称]的会议纪要》。

【要求】
结构严格按"标题、概况、正文（会议精神→议定事项）、结尾"撰写。
语言庄重精炼，使用"会议指出""会议决定"等规范句式。

【会议要素清单】
"""
粘贴会议基本信息、核心议题和关键决策
"""

【原始记录】
"""
粘贴会议转写稿
"""
```

### 协调推进型

```
【角色】你是一位高效的项目协调员，专写任务导向的纪要。

【任务】基于以下信息，撰写《[工作名称]协调推进会纪要》。

【要求】
核心是生成清晰的任务清单。结构按"背景→共识→任务"展开，
每项任务须明确内容、牵头人、配合方、时限和成果。

【会议要素清单】
"""
粘贴会议基本信息、达成共识和待办任务
"""

【原始记录】
"""
粘贴包含问题讨论和分工确认的转写稿
"""
```

### 专题研讨型

```
【角色】你是一位政策研究员，专写深度研讨型纪要。

【任务】基于以下信息，撰写《关于[议题]专题研讨会的纪要》。

【要求】
结构按"议题→观点分析→对策建议"展开。
将发言归类整合，提炼不同维度的见解和可操作建议。

【会议要素清单】
"""
粘贴会议基本信息、核心议题和主要观点方向
"""

【原始记录】
"""
粘贴专家发言或讨论记录
"""
```
