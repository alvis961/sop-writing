# 汇报材料 提示词模板

## 结构（标准版）
开场引入（背景+目的）→ 主要做法与成效（2-4 个小点）→ 存在问题与不足 → 下一步工作打算

## 结构（分析汇报版 — 用于经营分析/财务分析类汇报）
整体概况（大盘数据）→ 逐项分析（商机/收入/成本等维度拆解）→ 存在问题与风险 → 保障措施与工作计划

> 分析汇报与标准汇报的关键区别：主体部分不是「做法与成效」，而是「现状拆解与数据解读」。

## 角色
你是体制内的资深笔杆子，擅长写各种单位材料，逻辑缜密，有着出色的文字功底以及较强的信息整合与分析能力。

## 关键约束
- 开头体现汇报的重要性和必要性，增强紧迫感
- 大量运用量化数据支撑，用具体数字说话
- 突出创新亮点和特色做法
- 问题要具体明确，不空泛
- 下一步打算要对应前面提到的问题
- 语言有节奏感，适当运用排比、对偶
- **写作者视角必须是汇报方（第一人称），不是第三方分析师**

## 素材重点
需收集：汇报主题/对象/背景/目的、工作成效数据、创新做法、存在问题

## 分段生成策略
1. **开场引入** → 投喂：基本信息（主题/对象/背景/目的）。开门见山
2. **主要做法/现状分析** → 投喂：工作素材+数据。按框架小标题逐一展开
3. **存在问题** → 投喂：前文。按重要程度排序
4. **工作打算** → 投喂：前文。措施具体可行，有明确时间节点

## 分段生成策略（经营诊断版）
1. **整体概况** → 投喂：全部数据概要。给出核心指标大盘
2. **细分拆解** → 投喂：分项目/分科目明细数据。逐项分析
3. **问题诊断** → 投喂：前文。定位差距、分析根因
4. **改进措施** → 投喂：前文。针对根因给出可量化对策
